# Execute in development

1. Clone repo
2. Execute
```
yarn install
```
3. Must have Nest CLI installed
```
npm i -g @nestjs/cli
```

4. Build the data base
```
docker-compose up -d
```

## Stack 

* MongoDB
* Nest JS
